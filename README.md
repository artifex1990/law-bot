# Legal Consultation Bot 🤖⚖️

Универсальный чат-бот для записи на юридические консультации. Создан по ТЗ из `tz.md`.

## Возможности

- ✅ **16 направлений** юридических услуг
- ✅ **Telegram** (MVP) — готов к использованию
- ✅ **Алгоритмы в YAML** — легко добавлять и изменять сценарии
- ✅ **SQLAlchemy ORM** — работа с БД через модели
- ✅ **SQLite** по умолчанию — запуск без настройки PostgreSQL
- ✅ **Удобный UX** — кнопки, подсказки, понятные сообщения

## Быстрый старт

### 1. Установка

```bash
cd legal_bot_app
poetry install
# или: pip install -r requirements.txt
```

### 2. Настройка

Создайте файл `.env` (скопируйте из `env.example`):

```bash
cp env.example .env
```

Минимально нужен **токен Telegram-бота**:

```
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
```

Получить токен: [@BotFather](https://t.me/BotFather) → `/newbot`

### 3. Запуск

```bash
poetry run python -m src.main
# или: python -m src.main
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать консультацию |
| `/restart` | Начать заново |
| `/help` | Справка |
| `/cancel` | Отменить текущее действие |

## Добавление нового алгоритма

1. Создайте `src/scenarios/algorithms/направление.yaml`
2. Используйте структуру из `bankruptcy.yaml` или `family.yaml`
3. Добавьте кнопку в `main.yaml` в шаг `direction_selection`
4. Перезапустите бота

## База данных

По умолчанию используется **SQLite** (`legal_bot.db` в корне проекта).

Для **PostgreSQL** укажите в `.env`:

```
DATABASE_URL=postgresql+asyncpg://user:password@localhost:5432/legal_bot_db
```
